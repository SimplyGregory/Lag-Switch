A Python-based network lag simulator that temporarily blocks outbound traffic using Windows Firewall rules.

A lag switch is a network tool that intentionally disrupts network traffic to simulate poor internet conditions. This implementation uses Windows Firewall to temporarily block outbound connections, creating artificial network lag or disconnection scenarios.
